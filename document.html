Install SalakCMS

Untuk cara menginstall SalakCMS, pertama silahkan mendownload file SalakCMS disini Lalu setelah selesai terdownload, silahkan extrak file .zip nya.
Seluruh file SalakCMS berada dalam folder /dist_cms , Upload seluruh isi folder /dist_cms ke Hosting atau ke Localhost.
Jangan lupa atur permission folder /Contents dan semua subfoldernya menjadi 777 .
Setelah itu anda dapat mencoba dengan mengakses http://localhost/ atau http://domainanda.com

Konfigurasi SalakCMS

Untuk pengaturan file konfigurasi terdapat pada folder /Cores/Configure.php
Silahkan atur sesuai keinginan.

$config['domain'] = "http://namadomain/";
$config['theme'] = "nama folder tema ....";
$config['limit'] = "jumlah limit ....";
$config['sitename'] = "nama situs ....";
$config['description'] = "deskripsi ....";
$config['copyright'] = "footer ....";

Cara Membuat Postingan

Pertama buat sebuah file dengan format .post menggunakan PC/Smartphone, setelah itu buka file tersebut menggunakan notepad atau apa saja, dimana anda dapat menuliskan di dalam file itu.
Untuk membuat postingan berikut formatnya:

{%title%} Judul Postingan
{%date%} Tanggal (untuk penulisan tanggal menggunakan format yyyy-mm-dd , contoh 2017-01-31)
{%author%} Nama penulis
{%content%} Tulis isi kontent disini.
Setelah selesai anda dapat mengunggah file tersebut ke dalam Hosting masukkan ke folder /Contents/posts/disini

Cara Membuat Menu

Hampir sama dengan cara membuat postingan, untuk membuat menu halaman silahkan untuk membuat file dengan format .nav , lalu buka file tersebut.
Kemudian untuk cara pengisian juga sama dengan cara membuat postingan sebagai berikut:

{%menu%} Isi nama menu
{%title%} Judul halaman menu
{%content%} Konten halam menu
Jika sudah selesai , silahkan di unggah ke dalam Hosting masukkan ke dalam folder /Contents/navbars/disini

Cara Menambahkan Widget

Untuk menambahkan widget hampir sama dengan menambahkan postingan dan menambahkan menu.
Buat sebuah file dengan format .wid lalu buka file tersebut kemudian isikan dengan format seperti berikut:

{%title%} Judul widget
{%content%} Isi konten widget
Setelah selesai kemudian unggah ke dalam Hosting masukkan ke folder /Contents/widgets/disini
